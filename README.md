# FinMent
FinMent is an AI-powered personal finance mentor that helps users plan investments, calculate FIRE goals, and receive personalized financial guidance.

---

## **Features**

* FIRE Planner
  Calculate retirement corpus and required monthly SIP.

* AI Financial Advisor
  Generate personalized financial suggestions.

* Money Health Score
  Evaluate financial health based on savings, income, and expenses.

* Local Data Storage
  Stores user inputs using SQLite.

* Simple Web Interface
  Built with HTML, CSS, and JavaScript.

---

## **Tech Stack**

| Layer     | Technology                             |
| --------- | -------------------------------------- |
| Frontend  | HTML, CSS, JavaScript                  |
| Backend   | Flask (Python)                         |
| AI Layer  | FastRouter API (Claude Sonnet 4.0)     |
| Database  | SQLite                                 |
| Libraries | NumPy, Pandas                          |

## **Setup Instructions**

### **1. Clone the Repository**

```
git clone https://github.com/your-username/finment.git
cd finment
```

---

### **2. Create Virtual Environment (Recommended)**

```
python -m venv venv
```

Activate the environment:

**Windows**

```
venv\Scripts\activate
```

**Mac/Linux**

```
source venv/bin/activate
```

---

### **3. Install Dependencies**

```
pip install -r requirements.txt
```

If requirements.txt is not available:

```
pip install flask numpy pandas transformers torch
```

---

### **4. Run the Application**

```
python app.py
```

---

### **5. Open in Browser**

```
http://127.0.0.1:5000
```

## **How It Works**

### **FIRE Calculation**

Retirement Corpus = Annual Expenses × 25

Monthly SIP is calculated using compound interest formulas.

---

### **AI Advice Generation**


Generates financial suggestions based on user inputs

---

### **Financial Health Score**

Calculated based on:

* Savings ratio
* Emergency fund coverage
* Income vs expenses
